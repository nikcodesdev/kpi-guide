⚠️ Общие запреты при включении в одну РН

❌ ЗАПРЕЩЕНО:

1. Включать несколько комплексных пакетов настроек 2. Включать однотипные услуги 3. Включать несколько видов сборки ПК 4. Включать услуги, не имеющие отношения к устройству
📌 Важное правило (п.4):
Если у покупателя есть потребность в услугах для разных устройств — нужно выписать отдельную РН для каждого устройства.
Например: с ПК нельзя включать услуги для смартфона/планшета в одну РН.
